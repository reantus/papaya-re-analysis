| Latin_square_group              | no               | int          | Grouping number to show a category for counterbalancing the trials                                                                       | 1 - 6                                                                                                                       |
| target_type                     | yes              | character    | Whether a target was an actual word or not for a trial                                                                                   | word or nonword                                                                                                             |
| target                          | yes              | character    | The actual target word shown in the trial                                                                                                | example: vest                                                                                                               |
